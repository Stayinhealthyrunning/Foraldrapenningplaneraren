# Foraldrapenningplaneraren
**Föräldrapenning-planeraren – ett smart verktyg för att planera föräldraledigheten**

Det här är ett kostnadsfritt planeringsverktyg som hjälper nyblivna föräldrar att planera och optimera sin föräldraledighet. Verktyget räknar på föräldrapenning, SGI-dagar, lägstanivådagar, netto- och bruttoersättning samt hjälper till att fördela dagarna så jämnt som möjligt över hela ledighetsperioden.

Med verktyget kan du:

* planera för en eller två föräldrar
* ange SGI, skattesats och kvarvarande dagar
* se ersättning både före och efter skatt
* automatfördela dagarna för jämnast möjlig månadsersättning
* ta hänsyn till regler om 180 SGI-dagar, lägstanivådagar och SGI-skydd
* lägga till extra personer som kan ta emot överlåtna dagar
* spara flera olika scenarier och jämföra olika upplägg
* skriva ut en tydlig och överskådlig kalender

Verktyget är byggt för att göra det enklare att förstå hur olika val påverkar ekonomi, ledighet och dagförbrukning. Det ersätter inte Försäkringskassans information, men kan vara ett bra stöd när man vill planera smartare och få bättre överblick.
